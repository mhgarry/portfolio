My Story:

I'm Matthew Garry, and I am a full stack web developer focused on creating exceptional web services and experiences. I'm currently attending a full stack web development bootcamp through Rutgers University. I'm well versed in a multitude of web development and design principles, tools, and frameworks with a passionate focus on using CSS, HTML, and Javascript to create beautiful, seamless, and secure web applications. I have experience using Agile software development and working with teams. I particularly excel at creating applications that are both beautiful and functional as shown in my examples below. I used Tailwind to create the MyDictionary project. I used Adobe Illustrator to create the logo for the project. I ultimately pulled everything together in the MyDictionary application to create an amazing user experience.

In the Tasks application I have featured below I mainly worked on the backend of the project. I worked inside of Node.js and I used express to create the project server. I used MySql to create the database for the application which stores user information as well as task information. I used Sequelize to create and seed dynamic tables for the database. I used passport for the authentication process of the Tasks web application to make sure that Tasks was a secure application. I used ESlint to refactor the application. I also managed the github repository.
